# Storybook

## Requirements

- Node 14+
- NPM <= 6

## Stacks

- Next.js + TypeScript
- Chakra UI
- styled-components
- Tailwind CSS

## Dev

```sh
npm run storybook
```

## Deployment

```sh
npm run build-storybook
```

## Note

`@chakra-ui/storybook-addon` seems to mess storybook docs up, so the docs tab is currently disabled.
